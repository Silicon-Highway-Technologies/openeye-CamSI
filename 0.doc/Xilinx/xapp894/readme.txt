2. OVERVIEW

This readme describes the accompanying files in the zip file which are associated with the MIPI HW demonstration platform.  

The purpose of these files is to assist in the design and simulation of a high speed, low cost MIPI interface.  





3. SOFTWARE TOOLS AND SYSTEM REQUIREMENTS

The accompanying files were successfully run on HyperLynx, ver 9.0

6. OTHER INFORMATION (OPTIONAL) 

File definitions : 

KFMC_VCC18_MIPI_TX_2_RX.ffs : Hyperlynx Free Form Schematic file that is shown in the picture in the word document.  This is the simulation model.
KFMC_VCC18_MIPI_TX_2_RX.pjh : Hyperlynx project file.
sim_model.pdf : schematic capture of simulated design in Adobe pdf format. 
Schematic Prints FMC Card Final.pdf: Schematic of FMC card which is used for MIPI demonstration platform.
